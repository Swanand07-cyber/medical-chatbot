# Medical Chatbot

A Spring Boot based Medical Chatbot that predicts possible diseases based on symptoms entered by the user.

## Features

- Search disease by symptoms
- Display disease information
- Show precautions
- Show risk factors
- Show medicines
- User-friendly interface
- MySQL database integration

## Technologies Used

- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- MySQL
- HTML
- CSS
- Maven

## Project Structure

src
├── controller
├── model
├── repository
├── service
├── resources
│ ├── templates
│ ├── static
│ └── data

## How to Run

1. Clone repository
2. Create MySQL database
3. Update application.properties
4. Run MedicalChatbotApplication.java
5. Open browser

http://localhost:8080/chat
